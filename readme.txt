WP Isotope is a plugin, made the WordPress way.

== Description ==

Did you wnat to first to develop your site and want to make portfolio section firster? So use WP Isotope to make firster to develope your work. It is focused on ease of integration, ease of use, modern web standards, and performance.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WP Isotope'
3. Activate WP Isotope from your Plugins page. Now create a page and use [wp_isotop] shotcode and published.

= From WordPress.org =

1. Download WP Isotope.
2. Upload the 'WP Isotope' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WP Isotope from your Plugins page. (You will be greeted with a Welcome page.)
